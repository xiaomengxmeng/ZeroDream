[Medium] 不同路径 (62)

🤔 我的原始思路

vector<vector > dp (grid.size(),vector(grid[0].size()),0);
dp[i][0] =dp[0][j] =1;
走到 (i,j) 可以从 上方和左边来
dp[i][j]=dp[i-1][j]+dp[i][j-1];

🔍 我的思考过程

  • dp[i][j] 表示走到 (i,j) 的路径数
  • 第一行（一直右移）和第一列（一直下移）都只有 1 种路径
  • 转移方程：dp[i][j] = dp[i-1][j] + dp[i][j-1]
  • 最终返回 dp[m-1][n-1]

💡 对比收获

  • 一维 dp 扩展到二维：dp[i] → dp[i][j]
  • 边界处理：第一行/列固定为 1
  • 核心思想和 70 题一样是求和，只是多了一个维度

[Medium] 不同路径 II (63)

🤔 我的原始思路

如果 (i, j) 是障碍物 dp[i][j]=0;
不能走
第一行/列遇到障碍物后，后面的格子怎么处理
后面的格子全为0
起点和终点本身如果是障碍物呢 0条路径

🔍 我的思考过程

  • 障碍物位置 dp[i][j] = 0（走不通）
  • 第一行/列遇到障碍物后，后面全为 0（用 break）
  • 起点或终点是障碍物 → 直接 return 0
  • 非障碍物位置正常转移：dp[i][j] = dp[i-1][j] + dp[i][j-1]

💡 对比收获

  • 与 62 的关系：62 是 63 的”无障碍”特例
  • break 处理边界障碍物很简洁 — 障碍物后面的格子保持默认 0
  • 本质还是求和型 DP，多了个条件判断

[Medium] 完全平方数 (279)

🤔 我的原始思路

dp[0]=0
dp[i]=dp[i-j*j]+1;

🔍 我的思考过程

  1. 状态定义：dp[i] 表示凑成 i 需要的最少完全平方数个数
  2. 初始想法分析：dp[i] = dp[i - j²] + 1 方向正确，但需要取最小值 — 每个 i 有多种 j 的选择
  3. 初始化：最坏情况全用 1²，所以 dp[i] = i
  4. 取最小值：外层 i 从 1→n，内层 j 从 1→√i，对所有 dp[i - j²] + 1 取 min
  5. 边界陷阱：内层条件用 j*j <= i 而不是 <，否则 i 本身就是平方数时（如 4=2²）会漏判最优解

💡 对比收获

  • 求和 DP vs 最值 DP：70 是 dp[i] = dp[i-1] + dp[i-2]（累加），279 是 dp[i] = min(dp[i-j²] + 1)（取最小）
  • 选择空间：70 只有 2 种选择（减 1 或 2），279 有 √n 种选择（减所有平方数）
  • 框架统一：可抽象为 dp[i] = 聚合(选法 dp[i - 选法] + 代价)，聚合可以是 +/min/max

[Medium] 不同路径 (62)

🎯 核心思路

二维 DP 入门：dp[i][j] 表示走到 (i, j) 的路径数。第一行/列只有一种走法，其他格从上方或左边过来。

🔑 关键代码片段

📚 学到的知识点

  • 二维 DP 模板：dp[i][j] + 边界初始化 + 双循环
  • 边界思维：第一行只能从左边来（1种），第一列只能从上面来（1种）

  • ❌ 转移方程误加 +1：dp[i][j] = dp[i-1][j] + dp[i][j-1] 就对了，不需要 +1
  • dp 大小是 m×n，返回 dp[m-1][n-1]
  • 与 70 题本质相同：都是求和，只是 70 是一维，62 是二维

[Medium] 完全平方数 (279)

🎯 核心思路

一维最值型 DP：dp[i] 表示凑成 i 需要的最少完全平方数个数。对每个 i，尝试减去所有可能的平方数 j²，取最小值。本质和爬楼梯一样是”减一个值看之前的状态”，但选择空间更大（√n 种），且从求和变成了求最小值。

🔑 关键代码片段

📚 学到的知识点

  • 最值型 DP 模板：dp[i] = min/max over 所有选择 (dp[i - 选择] + 代价)
  • 初始化为上界：最值问题中 dp[i] 先设为可能的最大值（这里是 i），再不断取小
  • “减一个值”递推结构：70 减 1/2 → 279 减 j²，框架一致，选择空间不同

⚠️ 易错点 & 反思

  • ❌ 内层循环条件用 < 而不是 <=，会漏掉 i 本身就是完美平方数的最优解（dp[4] = 1）
  • 必须遍历所有 j² ≤ i，不只是部分平方数
  • dp[0] = 0 作为 base case，当 i = j² 时走 dp[0] + 1 = 1

🔗 关联题目

    1. Climbing Stairs（同结构，求和型 + 固定选择）
    1. Coin Change（同类：最值型 DP，选择硬币面额）
