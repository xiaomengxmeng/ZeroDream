7.1 常见问题

问题 原因 解决方案
Host key verification failed SSH known_hosts 未配置 运行 ssh-keyscan github.com >> ~/.ssh/known_hosts
Permission denied (publickey) Deploy Key 未配置或无效 检查 Secrets 中的 DEPLOY_KEY
fatal: could not read Username HTTPS 方式部署但无认证 改用 SSH 方式或配置 Personal Access Token
文章未生成 源文件不存在 检查 LeetCode-Solutions 仓库是否有对应日期的文件
封面图重复 随机算法问题 已改为确定性随机，同一日期同一封面
